Exemplos de Ciencia de dados

Data Science ou Ciência de Dados é um estudo muito disciplinado com relação aos dados e demais informações inerentes à empresa e as visões que cercam um determinado assunto.

Em resumo é uma ciência que visa estudar as informações, seu processo de captura, transformação, geração e, posteriormente, análise de dados. A ciência de dados envolve diversas disciplinas. São elas:

  • Estatística
  • Computação
  • Conhecimento do negócio
  • Matemática

Scientist ou Cientista de Dados. Existe uma pequena confusão sobre os motivos de não definirmos o Cientista de Dados como Data Scientist.

O Cientista de dados é basicamente um profissional multidisciplinar, que tem como responsabilidade colocar em pratica o processo de Data Science, conforme explicamos anteriormente.

Nesse caso, esse profissional é responsável em transformar dados em informações ou produtos de informação dentro de uma empresa ou corporação.

Outro ponto é que ele também tem como missão realizar a formulação de problemas, selecionar modelos de simulação e estatística e realizar a entrega dos produtos de dados.

Mas porquê a Ciência de Dados é tão fundamental para as empresas?

Por mais que você ainda não tenha se atentado a esse fato, grande partes dos negócios já faz parte do mercado de dados e informações.

A expectativa é que em 2017 sejamos cerca de 5 bilhões de mobile phones conectados, que serão responsáveis por produzir um fluxo gigantesco e constante de informação digital.
